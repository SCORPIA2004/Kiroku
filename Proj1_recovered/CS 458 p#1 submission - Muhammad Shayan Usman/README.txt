Login app is live on this link: https://login-app-seven-zeta.vercel.app/

Sample user creds are as follows:  { email: "test@example.com", password: "password123" }.
Alternatively, you can also choose to sign in with any google or facebook account.

----------------------------------------------------------------------------------------------------------------

- TO RUN TEST SUITE:

Source code for testing suite is inside test_login.py. To run it,
pip install selenium webdriver unittest
python test_login.py

The testing script will run through all test cases and give OK if all pass.

----------------------------------------------------------------------------------------------------------------

- TO RUN THE login-app:

cd login-app
npm install
npm run start

Application will start on localhost:3000
